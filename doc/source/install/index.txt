=============================
 Getting Started with Schevo
=============================

Follow these steps to get a Schevo environment ready on your computer, ready
for you to begin developing Schevo applications and following the `tutorials
<../tutorial/index.html>`__.

You can use Schevo in whatever way you choose, but these pages describe the
tools and techniques that Schevo documentation assumes you are using.
