==================
 Schevo Tutorials
==================

To help you learn Schevo, use these tutorials to introduce you to
various concepts and features of Schevo.

The tutorials intend to take you from starting a new project all the
way to turning it into a usable application.


Movie Reviews
=============

- The `Movie Reviews tutorial <moviereviews/>`__ shows you how to use
  Schevo to build a movie rating database.
